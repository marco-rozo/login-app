# login_app

Projeto flutter com exemplo de autenticação.
- GetX com gerenciador de estado
- GetConnect para requisições http
- GetStorage para salvar key-value na memoria


## Getting Started

install
```
$ flutter pub get
```

run 
```
$ flutter run
```

run in release mode
```
$ flutter run --release
```
